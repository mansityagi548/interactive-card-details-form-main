# Interactive Credit Card Form 💳 : 

A fully interactive credit card details form with **real-time card preview**, form validation, and smooth animations. 

## ✨ Features : 

- **Real-time Card Preview** – Updates instantly as you type
  
- **Responsive Design** – Looks great on desktop and mobile
  
- **Form Validation** – Proper error messages for each field

- **Interactive Card Flip** – Shows CVV on the back when focused
  
- **Clean & Modern UI** – Matches the original design perfectly
  
- **Accessibility Focused** – Keyboard navigation and screen reader friendly

## 🛠️ Technologies Used

- **HTML5**
- **CSS3** (Flexbox & Grid)
- **Vanilla JavaScript** (DOM manipulation & validation)
